- quote: # 5.1主要咨询方法与适用原理
主要咨询方法为焦点解决短期咨询和治疗技术。焦点解决短期疗法是一种以来访者为中心的治疗取向，认为来访者是自己问题情境的专家[2]。该疗法的主要观点体现在两个方面：一方面，他们相信人本身已拥有解决问题的潜能，但是当来访者寻求协助时，往往为眼前的问题所困扰，在以往无效的方法里不断打转。因此咨询师要善用来访者已有的潜能，并且加以发挥。另一方面，该疗法把治疗的焦点放在探讨问题不发生的状况（即“例外情况”），一点点增加这个“例外”的部分，整个内部系统的改变就发生了[3]。

# 5.2咨询设置
根据评估结果及初步确定的咨询目标，咨询师与来访者签订知情同意书、保密协议。这将明确双方在咨询过程中的责任、权利和义务，并向来访者详细说明涉及的保密原则和保密例外情况。咨询时间及咨询次数商定如下：咨询时间，每周1次，每次 $4 0 { \sim } 5 0 \mathrm { m i n }$ ；咨询4次。
