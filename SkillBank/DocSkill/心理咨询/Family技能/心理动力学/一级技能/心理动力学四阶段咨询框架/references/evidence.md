在初次咨询中，咨询师和来访者确定了每周1次的咨询设置。本个案自2023年2月第一次咨询伊始，至2023年10月（暑期暂停）结束，一共进行了28次，每次咨询时长50～$6 0 \mathrm { { m i n } }$ 。咨询对学生免费。

3.3咨询思路
心理动力学强调早年童年经历对人格塑造的重要性，通过探索来访者当前的应对（防御）方式来追溯痛苦的根源，主张无意识的心理活动可以影响有意识的思想、情感和行为。心理动力学取向的心理咨询过程，并不直接解决来访者的现实问题，而是强调让来访者的无意识意识化、支持来访者薄弱的自我功能以及在这个过程中帮助其发展新的思维和情感模式。在心理动力学咨询的第一个阶段，咨询师对来访者进行评估及个案概念化，并对咨访关系的匹配性做初步的考量；在第二个阶段，咨询师需要运用基础的倾听和咨询技术，和来访者建立关系同盟，为后续咨询的深入奠定关系基础，并让来访者的情绪在这个阶段得到初步的宣泄；在关系建立的前提下，咨询师需要在第三个阶段，让来访者意识到其当下某些重复的思维、情感或行为模式，是受其童年经验的影响而形成的，并通过向来访者分享咨询师“自我”的调节、合理化、哀悼等功能，引导来访者用更适应的方式去应对当前遇到的困境；在最后一个阶段，咨询师帮助来访者一起，将新的思维及情感的应对模式进行内化，并对未来可能遇到的问题进行提前演练，增强其离开咨询后的自主应对能力。

在充分沟通的前提下，来访者小云同意咨询师在心理动力学框架下对其展开咨询。
