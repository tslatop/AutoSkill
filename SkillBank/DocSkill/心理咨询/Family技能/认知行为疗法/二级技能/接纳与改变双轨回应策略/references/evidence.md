4.2.3解决问题并提高自尊阶段：在完成团体技能训练后，以技能为基础帮助来访者解决还未被完全接受的部分，并帮助其提高生活质量。本阶段形式采用个体治疗，共4次，每周1次，每次 $5 0 \mathrm { { m i n } }$ 。

第一次治疗：分为两个阶段，第一阶段用以了解来访近期状况与问题。第二阶段，根据来访者需求，以正念观呼吸的方式解答来访关于正念技能的困惑，具体分为以下步骤。①准备工作：邀请来访者调整坐姿，并保持觉察。 $\textcircled{2}$ 技巧指导。指导语如下：“现在轻轻地闭上眼，将注意力从过去带到当下，将觉察带到呼吸上，去感受下空气经过鼻腔带给你的感受。不用刻意控制呼吸节奏。”约2min后继续引导：“或许注意力会偏移，没有关系，感谢自己，然后轻轻地把注意力拉回到当下。不要排斥，也不要抗拒，保持对呼吸的觉察。”重复引导两次。 $\textcircled{3}$ 练习后，来访者表示会被外面的一些声音吸引，容易走神，觉得自己做得不好。治疗师回应：“并不是走神引起的情绪，而是评价引起的情绪。正念所引导的不仅仅是觉察，也是一种不评判的态度。想要打破批判的模式，你的大脑自然会产生反抗。坚持练习，你的痛苦和烦恼会慢慢地减少。” $\textcircled{4}$ 咨询师陪来访者回顾练习的细节问题，一起制定练习计划，并邀请来访者以正念的方式观察自己当下的生活，下周再讨论。

第二次治疗：治疗开始时，来访者主动分享在过去一周中自己的记录和感受。针对来访者的反馈，治疗师采用接纳和改变的策略进行回应，引导来访者接纳当下的自己，对评价不去控制或驱赶，只是将注意力放到当下。同时，澄清来访者在母女关系中的目标，并与来访者讨论目标效能与自尊效能以帮助来访者达到其人际目标。部分逐字稿如下。

小D：我开始用正念的方式去对待自己的生活，感觉更轻松了一些。很多时候我能够去观察到我的一些冲动了，这让我更有掌控感。但我还是会忍不住想如果自己更完美一些，就像妈妈希望我成为的那样，妈妈一定会更喜欢我。
